Как начать

  1. Установи Cursor — курс работает как интерактивный Canvas.
  2. Склонируй репозиторий с GitHub или скачай ZIP.
  3. Положи файл canvas/aml-kyc-training.canvas.tsx в папку canvases/ своего workspace в Cursor.
  4. Открой файл — курс запустится рядом с чатом. Прогресс сохраняется локально.
